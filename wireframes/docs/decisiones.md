# Guía de Estilos – Sistema Académico Accesible

## 1. Paleta de Colores

Se seleccionaron colores con contraste mínimo 4.5:1 (WCAG AA).

| Uso | Color | Hex |
|------|--------|------|
| Primario | Azul Oscuro | #1E3A8A |
| Secundario | Azul Claro | #2563EB |
| Fondo | Blanco | #FFFFFF |
| Fondo Secundario | Gris Claro | #F3F4F6 |
| Texto Principal | Gris Oscuro | #374151 |
| Error | Rojo | #DC2626 |

---

## 2. Tipografía

Fuente utilizada: Inter

- Texto base: 16px
- Títulos principales: 24px
- Subtítulos: 20px
- Botones: 16px bold

Se permite escalado hasta 200% sin pérdida de funcionalidad.

---

## 3. Espaciado

Sistema basado en múltiplos de 8px:

- 8px
- 16px
- 24px
- 32px

Esto garantiza consistencia visual.

---

## 4. Componentes

### Botones
- Altura mínima: 44px
- Estados: normal, hover, focus, active, disabled
- Indicador visible de foco

### Formularios
- Labels visibles
- Mensajes de error descriptivos
- Campos con borde rojo en error

### Navegación
- Breadcrumbs visibles
- Skip link
- Menú accesible por teclado
