# Sistema Académico Accesible

Proyecto académico orientado al diseño de un sistema académico web con enfoque en accesibilidad, aplicando principios de Ingeniería de Software (SWEBOK) y estándares WCAG 2.2 nivel AA.

---

## Objetivo del Proyecto

Diseñar un sistema académico accesible que permita a estudiantes, profesores y administradores gestionar información académica garantizando:

- Accesibilidad universal
- Navegación por teclado
- Alto contraste
- Diseño centrado en el usuario
- Cumplimiento de WCAG 2.2 AA

---

## Alcance

El proyecto incluye:

- Wireframes en papel
- Prototipo de alta fidelidad en Figma
- Guía de estilos accesible
- Documentación de decisiones de diseño
- Diagrama estructural del sistema
- Video demostrativo

No incluye implementación en código (solo diseño y documentación).

---

## Usuarios Identificados

1. Estudiante
2. Profesor
3. Administrador
4. Estudiante con necesidades de accesibilidad (baja visión o dislexia)

---

## Principios Aplicados (SWEBOK)

Este proyecto aplica conocimientos de:

- Ingeniería de Requisitos
- Diseño de Software
- Calidad del Software
- Usabilidad y Experiencia de Usuario
- Arquitectura de Software

---

## Cumplimiento de Accesibilidad (WCAG 2.2 AA)

Se consideraron los siguientes criterios:

- 1.4.3 Contraste mínimo
- 2.1.1 Navegación por teclado
- 2.4.3 Orden del foco
- 2.4.7 Foco visible
- 3.3.1 Identificación de errores
- 4.1.2 Nombre, función y valor

## Prototipo en Figma

- Componentes reutilizables

- Navegación accesible
